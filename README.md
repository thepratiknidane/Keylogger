# Keylogger

This Keylogger program record the every key stroke and save in log files.

<h3>Requirements:</h3>
1. Python 3 (pynput)

Installation:

    git clone https://github.com/thepratiknidane/Keylogger.git

    cd Keylogger

    python3 main.py

It's Record the keystrokes until you stop the execution of program
